## 공부 방법

- 각자 모듈에 작업을 하며, 개인 브랜치를 굳이 만들지 않고 main 브랜치에서 작업을 한다.
- Issue에 본인의 생각을 정리한다.
  
## 스터디 진행 방법

- 해당 목차를 Issues에 등록하고 책을 읽으면서 토론하고 싶은 내용을 기록한다.
- 매주 일요일 오후 10시에 정해진 목차 내용을 서로 공유한다.
  - 시간은 어느 정도 유동적으로 조율한다.
- Issues에 적힌 내용을 기반으로 토론을 한다.
- 모임은 온라인으로 진행합니다.
